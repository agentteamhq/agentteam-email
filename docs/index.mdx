AgentTeam Email is an open-source email service designed so every AI agent in your system gets its own fully functional mailbox. You self-host the entire stack — there are no third-party mailbox providers and no shared inboxes. Mail flows in through Cloudflare Email Routing, gets archived to R2-compatible object storage, and is delivered outbound through ZoneMTA. A web application lets you administer mailboxes, review messages, and manage your Cloudflare integration, all from one place.

## What AgentTeam Email provides

- **Dedicated agent mailboxes** — each agent receives its own email address backed by a [WildDuck](https://wildduck.email/) IMAP server, keeping agent mail fully isolated from every other mailbox on your domain.
- **Inbound mail capture** — Cloudflare Email Routing forwards incoming messages to a Cloudflare Worker that normalises and stores each message before forwarding it into the WildDuck mailbox.
- **Raw message archival** — every inbound message is written to R2-compatible object storage as three files: `raw.eml` (the original RFC 5322 message), `edge.json` (Cloudflare envelope metadata), and `result.json` (processing outcome), giving you a complete, inspectable audit trail.
- **Outbound delivery** — agents send mail through ZoneMTA, which can relay via Cloudflare Email or Amazon SES, so you retain full control over your sending reputation and delivery pipeline.
- **Web admin interface** — an authenticated web application lets you create and manage mailboxes, inspect individual messages, and configure your Cloudflare Email Routing integration without touching the CLI.
- **Flexible self-hosting** — run the full stack on a single host with Docker Compose, or deploy to any Kubernetes cluster using the included Helm chart.
