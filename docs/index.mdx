AgentTeam Email is an email control plane for agent-owned mailboxes, inbound
message capture, and operator-managed mail infrastructure.

The current documentation focuses on the self-hosted deployment shape: the web
app, mail control service, Cloudflare Email Worker, archive storage, mail
services, and the operator-owned Worker ingress route.
